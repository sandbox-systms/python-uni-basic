# Modulo 08 - Projeto integrador

Este modulo junta variaveis, operadores, condicionais, loops, funcoes, tipos de dados, formatacao e tratamento de erros em um unico fluxo.

## Objetivo tecnico

Mostrar como os conceitos estudados nos modulos anteriores se conectam em um programa um pouco mais realista.

## Projeto da secao

### 1. `01-sistema_academico_integrado.py`

O que o projeto faz:
- recebe dados de um aluno
- valida entradas com utilitarios compartilhados
- calcula media e frequencia
- classifica a situacao final
- exibe um boletim formatado

Conceitos aplicados:
- `input()` e `print()`
- `try/except`
- funcoes
- dicionarios
- formatacao de strings
- validacao de faixa

## Como executar

```bash
python3 08-projeto-integrador/01-sistema_academico_integrado.py
```
