# 🐍 Python Uni Basic - Curso de Python para Iniciantes

Bem-vindo ao repositório de estudos de Python! Este projeto contém um curso completo de Python organizado em módulos numerados para facilitar o aprendizado progressivo.

## 📚 Estrutura do Curso

| Módulo | Tópico | Descrição |
|--------|--------|-----------|
| 01 | Hello World | Primeiro programa em Python |
| 02 | Variáveis | Declaração e uso de variáveis |
| 03 | Operadores | Operadores aritméticos, lógicos e mais |
| 04 | Condicionais | Estruturas if, elif, else |
| 05 | Loops | Estruturas for e while |
| 06 | Funções | Criação e uso de funções |
| 07 | Tipos de Dados | Strings, números, listas, dicionários |

## 🚀 Como Usar

1. Clone o repositório
2. Navegue até o módulo desejado
3. Leia o `README.md` do módulo
4. Execute os arquivos `.py` para praticar

```bash
# Exemplo de execução
python 01-hello-world/exemplo.py
```

## 📋 Pré-requisitos

- Python 3.6 ou superior instalado
- Um editor de texto ou IDE (VS Code, PyCharm, etc.)

## 🎯 Objetivo

Este material foi criado para ensinar os conceitos básicos de Python de forma prática e progressiva, com exemplos claros e exercícios para fixação do conhecimento.
