# Loops em Python

Esta pasta reúne exemplos simples de estruturas de repeticao e controle de fluxo em Python. O foco aqui e praticar `while`, `break`, `continue`, `elif` e uma simulacao de `switch case` usando dicionarios e funcoes.

## Topicos abordados

- `while True` para criar menus e repeticoes continuas
- `break` para encerrar um loop
- `continue` para pular uma iteracao
- `elif` para tratar multiplas condicoes
- simulacao de `switch case` em Python

## Arquivos da pasta

- `while-true.py`: mostra um menu simples que continua executando ate o usuario escolher sair
- `contiue-break.py`: demonstra como `continue` e `break` alteram o fluxo de um `for`
- `elif.py`: apresenta exemplos com `if`, `elif` e operadores logicos
- `switch-case.py`: simula um menu de opcoes usando funcoes e um dicionario

## O que voce vai praticar

Com esses exemplos, voce pode entender como repetir blocos de codigo, controlar a parada de um loop e organizar decisoes com varias condicoes. Esses conceitos sao muito usados em menus, validacao de entrada, jogos simples e pequenos programas interativos.

## Como executar

Entre na pasta `loops` e rode qualquer arquivo com Python:

```bash
python while-true.py
python contiue-break.py
python elif.py
python switch-case.py
```

## Requisito

- Python 3 instalado na maquina

## Observacao

Alguns exemplos usam `input()`, entao precisam ser executados no terminal para que voce possa digitar as opcoes e testar o comportamento dos loops na pratica.
