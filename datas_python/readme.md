# Estudo de Fundamentos de Python para a Universidade

## Descrição

Este repositório é um guia prático de **conceitos fundamentais de Python**, focando em algoritmos, resolução de problemas e aplicação prática da linguagem.

## 📚 Conteúdo Abordado

- **Dados**: Conceitos básicos e estrutura de dados
- **Manipulação de Dados**: Operações e transformações
- **Tipos de Dados**: Inteiros, strings, listas, dicionários, etc.
- **Funções Básicas de Input/Output**: Entrada e saída de dados no terminal

## 🎯 Objetivo

Desenvolver um programa funcional mínimo que integre todos os conceitos aprendidos durante a sessão, aplicando os conhecimentos através da linha de comando (terminal).

## 📖 Como Usar

1. Clone o repositório
2. Navegue até a pasta `datas_python`
3. Execute os scripts Python com o comando: `python nome_do_arquivo.py`

## 🛠️ Pré-requisitos

- Python 3.7+
- Terminal/Console
